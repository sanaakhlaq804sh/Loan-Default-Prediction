# Loan-Default-Prediction
Machine Learning project to predict loan default using Logistic Regression.

## Project Overview
This project predicts whether a loan applicant will repay the loan or default using Machine Learning.

## Dataset Features
- Annual Income
- Loan Amount
- Interest Rate
- Credit Utilization
- Employment Status
- Loan Purpose
- And more...

## Steps Performed
- Data Cleaning
- Handling Categorical Variables
- Feature Scaling
- Train-Test Split
- Logistic Regression Model
- Model Evaluation

## Model Performance
The Logistic Regression model achieved good accuracy in predicting loan repayment status.

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Scikit-learn
